选择器

选择器的作用：可以帮助你精准的选择想要的元素

简单选择器

ID选择器

#id {
    color: red;
}

元素选择器

p {
    color: red;
}

类选择器

.class {
    color: red;
}

通配符选择器

* {
    color: red;
}

属性选择器

[属性名]: 选中所有具有该属性名的元素
[属性名=属性值]: 选中属性名=属性值的 该元素


更多知识链接：https://developer.mozilla.org/zh-CN/docs/Web/CSS/Attribute_selectors

伪类选择器 ( 目前先学习如下)

选中某些元素的某种状态

hover

鼠标悬停的状态

一般用于a元素上

a:hover{
    color: blue;
}

acitve

激活时的状态

a:active{
    color: blue;
}

link

超链接未被访问的状态

visited

超链接访问过后的状态

伪元素选择器 ( 目前先学习如下)

before & after

span::before{
    content: '《';
    color: red;
}
span::after{
    content: "》";
    color: blue;

选择器的组合

  • 并且的多个选择器组合： 直接写到一起
    span::before{

    }
  • 后代元素选则组合： ‘’ + 空格 + '’
    ul li {
        color: 'red'
    }
    .classOne .classTwo {

    }
    在用了classOne类样式所在的元素的子元素中，用了classTwo类样式的元素
  • 子元素的选择组合： '’ + ‘>’ + ‘’

只能选择子元素，不能选择子元素的子元素

    ul>li{
        color: red;
    }
  • 相邻兄弟元素 ‘_’ + ‘+’ + ‘元素’

相邻的下一个兄弟元素

  • 后面所有的兄弟元素 ‘_’ + ‘~’ + ‘元素’

该选择元素后面的所有的兄弟元素

只能选择后面的兄弟元素，选择不了前面的兄弟元素，因为css渲染机制的问题

选择器的并列

多个选择器，用逗号分割

p, h1, .classs {
    color: red;
}

应用场景，多个元素，有同一样式，为了更好的便于维护，可以选择选择器的并列写法
